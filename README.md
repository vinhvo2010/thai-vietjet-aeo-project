> **Answer Engine Optimization (AEO) Implementation Strategy for Thai Vietjet (th.vietjetair.com)**

## 📋 Project Overview

This repository contains the complete **AI-First Marketing & Answer Engine Optimization (AEO)** implementation strategy for **Thai Vietjet Air**, a low-cost carrier operating Boeing 737-8/MAX 8 fleet across Thailand, Vietnam, Australia, Japan, and Taiwan.

### 🎯 Objectives

- **Increase AI Search Visibility**: Optimize for ChatGPT, Perplexity, and AI-driven search engines
- **Boost Organic Traffic**: Target 30-50% increase in organic search traffic
- **Drive Direct Bookings**: Increase direct booking conversion by 25-40%
- **Enhance Brand Presence**: Establish Thai Vietjet as the leading AI-optimized airline in Southeast Asia
