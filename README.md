## 🚀 Chiến Lược Triển Khai AEO

### Giai Đoạn 1: Foundation (Tháng 1)

- [ ] Audit website hiện tại
- [ ] Cài đặt Schema Organization, FAQPage, Flight
- [ ] Thiết lập GA4 theo dõi AI referrals
- [ ] Tối ưu 5-10 trang chính (homepage, top routes)

### Giai Đoạn 2: Content Creation (Tháng 2-3)

- [ ] Tạo 15-20 trang route (Bangkok-Phuket, Bangkok-Chiang Mai, v.v.)
- [ ] Tạo 10-15 destination guides (Phuket, Chiang Mai, Da Nang, v.v.)
- [ ] Tạo 10-15 FAQ pages (baggage, check-in, SkyBOSS)
- [ ] Triển khai schema cho tất cả trang mới

### Giai Đoạn 3: Off-Page & UGC (Tháng 4-5)

- [ ] Xây dựng hiện diện Reddit & Quora
- [ ] Xây dựng backlink từ travel blogs
- [ ] Chạy Digital PR về đội bay Boeing 737-8 mới
- [ ] Khuyến khích customer reviews

### Giai Đoạn 4: Optimization (Liên Tục)

- [ ] Kiểm tra AI visibility hàng tháng
- [ ] Theo dõi AI referral traffic
- [ ] A/B test nội dung
- [ ] Cập nhật nội dung theo mùa

  "@context": "https://schema.org",
  "@type": "Flight",
  "departureAirport": {
    "@type": "Airport",

    "iataCode": "BKK"
  },
  "arrivalAirport": {

    "iataCode": "HKT"
