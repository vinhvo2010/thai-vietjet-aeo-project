1. Organization Schema

Usage: Add to homepage (th.vietjetair.com) in the <head> section

Purpose: Defines Thai Vietjet as an organization, establishing brand identity for search engines and AI systems

Key Properties:

  • @type: "Airline" (defines the organization type)
  • name: Official airline name
  • iataCode: "VZ" (airline code for Thai Vietjet)
  • url: Homepage URL
  • logo: High-resolution logo URL
  • sameAs: Social media profiles for authority
  • contactPoint: Customer service information
{
  "@context": "https://schema.org",
  "@type": "Airline",
  "name": "Thai Vietjet Air",
  "alternateName": "Thai VietJet",
  "iataCode": "VZ",
  "url": "https://th.vietjetair.com",
  "logo": {
    "@type": "ImageObject",
    "url": "https://th.vietjetair.com/images/logo.png",
    "width": 600,
    "height": 200
  },
  "description": "Thai Vietjet Air is a low-cost airline operating domestic and international flights from Thailand to Vietnam, Japan, Taiwan, and Australia.",
  "sameAs": [
    "https://www.facebook.com/thaivietjetair",
    "https://www.instagram.com/thaivietjetair",
    "https://twitter.com/thaivietjetair",
    "https://www.youtube.com/thaivietjetair",
    "https://en.wikipedia.org/wiki/Thai_VietJet_Air"
  ],
  "contactPoint": {
    "@type": "ContactPoint",
    "telephone": "+66-2-126-8999",
    "contactType": "Customer Service",
    "areaServed": ["TH", "VN", "JP", "TW", "AU"],
    "availableLanguage": ["Thai", "English", "Vietnamese"]
  },
  "foundingDate": "2014",
  "address": {
    "@type": "PostalAddress",
    "streetAddress": "Don Mueang International Airport",
    "addressLocality": "Bangkok",
    "addressCountry": "TH"
  }
}

2. Flight Schema

Usage: Add to individual route pages (e.g., Bangkok to Phuket page)

Purpose: Provides detailed flight information for AI systems to answer queries like "flights from Bangkok to Phuket" or "Thai Vietjet flight times"

Key Properties:

  • flightNumber: Complete with airline code (e.g., "VZ101")
  • departureAirport/arrivalAirport: Use IATA codes (BKK, HKT)
  • departureTime/arrivalTime: ISO 8601 format with timezone
  • provider: Link to airline organization
  • estimatedFlightDuration: ISO 8601 duration format
{
  "@context": "https://schema.org",
  "@type": "Flight",
  "flightNumber": "VZ101",
  "provider": {
    "@type": "Airline",
    "name": "Thai Vietjet Air",
    "iataCode": "VZ",
    "url": "https://th.vietjetair.com"
  },
  "departureAirport": {
    "@type": "Airport",
    "name": "Don Mueang International Airport",
    "iataCode": "DMK",
    "address": {
      "@type": "PostalAddress",
      "addressLocality": "Bangkok",
      "addressCountry": "TH"
    }
  },
  "departureTime": "2026-03-15T08:00:00+07:00",
  "departureGate": "A12",
  "departureTerminal": "1",
  "arrivalAirport": {
    "@type": "Airport",
    "name": "Phuket International Airport",
    "iataCode": "HKT",
    "address": {
      "@type": "PostalAddress",
      "addressLocality": "Phuket",
      "addressCountry": "TH"
    }
  },
  "arrivalTime": "2026-03-15T09:20:00+07:00",
  "arrivalGate": "3",
  "arrivalTerminal": "Domestic",
  "estimatedFlightDuration": "PT1H20M",
  "flightDistance": {
    "@type": "Distance",
    "value": "682",
    "unitCode": "KMT"
  },
  "aircraft": {
    "@type": "Vehicle",
    "name": "Airbus A320",
    "vehicleModelDate": "2020"
  },
  "boardingPolicy": "https://schema.org/ZoneBoardingPolicy",
  "mealService": "Snacks and beverages available for purchase",
  "webCheckinTime": "2026-03-14T08:00:00+07:00"
}

💡 Pro Tip: Add this schema to every route page. Create dynamic pages for all city pairs (Bangkok-Phuket, Bangkok-Hanoi, etc.) to capture long-tail search queries and AI recommendations.

4. FAQPage Schema

Usage: Add to FAQ pages, help center, and pages with Q&A content

Purpose: Powers Google's FAQ rich results and helps AI systems answer direct questions about your airline

Key Properties:

  • mainEntity: Array of Question objects
  • Question.name: The question text
  • acceptedAnswer: The answer content
  • Each question should have a clear, concise answer (under 300 characters for best results)
{
  "@context": "https://schema.org",
  "@type": "FAQPage",
  "mainEntity": [
    {
      "@type": "Question",
      "name": "What is Thai Vietjet's baggage allowance?",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "Thai Vietjet allows 7kg of cabin baggage (one main bag + one personal item). SkyBoss fares include 30kg checked baggage, Deluxe fares include 20kg, and Eco fares do not include checked baggage (can be purchased separately)."
      }
    },
    {
      "@type": "Question",
      "name": "How early should I arrive at the airport for my Thai Vietjet flight?",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "For domestic flights, arrive at least 90 minutes before departure. For international flights, arrive at least 2 hours before departure. Check-in counters close 40 minutes before departure."
      }
    },
    {
      "@type": "Question",
      "name": "Can I check in online for Thai Vietjet flights?",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "Yes, online check-in is available 24 hours before departure through th.vietjetair.com or the Thai Vietjet mobile app. You can select seats and download your boarding pass. Check-in closes 1 hour before departure."
      }
    },
    {
      "@type": "Question",
      "name": "What destinations does Thai Vietjet fly to?",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "Thai Vietjet operates domestic flights within Thailand (Bangkok, Phuket, Chiang Mai, Krabi, Khon Kaen) and international flights to Vietnam (Hanoi, Ho Chi Minh, Da Nang, Phu Quoc), Japan (Fukuoka), Taiwan (Taipei), and Australia (Sydney, Melbourne)."
      }
    },
    {
      "@type": "Question",
      "name": "What is Thai Vietjet's cancellation policy?",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "Cancellation fees vary by fare type. Eco fares have higher cancellation fees, while SkyBoss fares offer more flexibility. You can cancel up to 4 hours before departure through the website or call center. Refunds are processed within 7-10 business days."
      }
    },
    {
      "@type": "Question",
      "name": "Does Thai Vietjet offer inflight meals?",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "Meals are not included in the base fare. Passengers can pre-order meals when booking or purchase food and beverages onboard. SkyBoss passengers receive complimentary meals and drinks."
      }
    }

}

⚠️ Important: Each FAQ answer should be under 300 characters for best results in Google Rich Results. Focus on direct, actionable answers.

6. Offer Schema

Usage: Add to promotional pages, special deals, and flight booking pages

Purpose: Helps search engines and AI systems understand current pricing and special offers

Key Properties:

  • @type: "Offer" or "AggregateOffer" for multiple price points
  • price/lowPrice: Numeric price value
  • priceCurrency: ISO 4217 currency code
  • priceValidUntil: Expiration date for promotional pricing
  • availability: InStock, OutOfStock, PreOrder
{
  "@context": "https://schema.org",
  "@type": "Product",
  "name": "Thai Vietjet Flight: Bangkok to Phuket",
  "description": "Direct flights from Bangkok (Don Mueang) to Phuket with Thai Vietjet Air. Multiple daily departures available.",
  "brand": {
    "@type": "Brand",
    "name": "Thai Vietjet Air"
  },
  "offers": {
    "@type": "AggregateOffer",
    "lowPrice": 750,
    "highPrice": 2500,
    "priceCurrency": "THB",
    "priceValidUntil": "2026-12-31",
    "availability": "https://schema.org/InStock",
    "url": "https://th.vietjetair.com/flights/bangkok-phuket",
    "seller": {
      "@type": "Organization",
      "name": "Thai Vietjet Air"
    },
    "offerCount": 14
  }
}

// For promotional "Zero Dong" campaigns:

Implementation Best Practices

Technical Implementation

  • Add JSON-LD in <script type="application/ld+json"> tags
  • Place in the <head> section for best results
  • Can also be placed in <body> near relevant content
  • Multiple schema types can coexist on one page
  • Use schema generators to minimize syntax errors

Validation & Testing

  • Google Rich Results Test: search.google.com/test/rich-results
  • Schema.org Validator: validator.schema.org
  • Test on mobile and desktop
  • Monitor Google Search Console for errors
  • Re-validate after any schema updates

Maintenance

  • Update pricing weekly for promotional fares
  • Sync flight times with actual schedules
  • Keep FAQs current with policy changes
  • Add new routes as they launch
  • Monitor for Schema.org updates annually

Common Pitfalls to Avoid

  • Don't hide schema content that's not visible on page
  • Avoid mismatches between schema and visible content
  • Don't use placeholder or fake data
  • Never markup unavailable flights or outdated prices
  • Don't ignore validation warnings
